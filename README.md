# 🌟 About The Project

This project is an advanced **AI-powered Food Classification System** developed using modern **Deep Learning and Computer Vision techniques**.

The application can classify **34 different food categories** using multiple deep learning architectures such as:

- 🧠 Custom CNN
- ⚙️ VGG16
- 🚀 ResNet50

The system also integrates:

✅ Flask Web Application  
✅ Redis Database  
✅ Nutrition Recommendation System  
✅ Real-Time Prediction  
✅ Transfer Learning  
✅ Interactive AI Dashboard

# ⚡ Deep Learning Workflow

```text
📷 Upload Food Image
        ↓
🧠 CNN / VGG16 / ResNet50
        ↓
📊 Feature Extraction
        ↓
📈 Softmax Probability Prediction
        ↓
🔴 Redis Nutrition Retrieval
        ↓
🍽 Final Prediction + Nutrition Data
```

# 🔴 Redis Database Workflow

```text
food_nutrition.json
        ↓

        ↓
Redis Database Storage
        ↓

        ↓
Nutrition Data Display In UI
```

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/GanjiNagendhraPrasad/Food-Classification-Using-Deep-Learning-CNN-.git
```

## 2️⃣ Navigate To Project Folder

```bash
cd Food-Classification-Using-Deep-Learning-CNN-
```

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

## 4️⃣ Start Redis Server

```bash
redis-server
```

## 5️⃣ Load Nutrition Data Into Redis

```bash
python send_to_redis.py
```

## 6️⃣ Download Model Weights

🔗 Download Here:

https://github.com/GanjiNagendhraPrasad/Food-Classification-Using-Deep-Learning-CNN-/releases/tag/v1.0.0

## 7️⃣ Run Flask Application

```bash
python app.py
```
