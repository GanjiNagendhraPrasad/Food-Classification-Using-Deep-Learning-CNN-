# 🌟 About The Project

## 🚀 Food Classification Using Deep Learning CNN

This project is an advanced **AI-powered Food Classification System** developed using:

- 🧠 Deep Learning
- 👁️ Computer Vision
- 🌐 Flask Web Framework
- 🔴 Redis Database
- ⚡ Real-Time Prediction System

The application can classify **34 different food categories** using multiple deep learning architectures.

---

# 🔥 Core Features

✅ Multi Deep Learning Models  
✅ Real-Time Food Prediction  
✅ Flask Web Application  
✅ Redis Database Integration  
✅ Nutrition Information Retrieval  
✅ Modern Interactive Dashboard  
✅ Transfer Learning Models  
✅ Top-5 Prediction System  
✅ End-to-End AI Workflow  

---

# 🧠 Deep Learning Models Used

| Model | Description | Purpose |
|------|------|------|
| 🧠 Custom CNN | Built completely from scratch using Conv2D, MaxPooling & Dense Layers | Custom Feature Extraction |
| ⚙️ VGG16 | Transfer Learning using pretrained VGG16 model | Better Accuracy |
| 🚀 ResNet50 | Advanced Residual Neural Network | High Performance Classification |

---

# ⚡ Deep Learning Workflow

```text
📷 Upload Food Image
        ↓
🧠 CNN / VGG16 / ResNet50
        ↓
📊 Feature Extraction
        ↓
📈 Softmax Probability Prediction
        ↓
🔴 Redis Nutrition Retrieval
        ↓
🍽 Final Prediction + Nutrition Data
